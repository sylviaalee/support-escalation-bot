# Two-Factor Authentication (2FA)

## Enabling 2FA
1. Go to **Settings > Security > Two-Factor Authentication**
2. Choose your method: Authenticator app (recommended) or SMS
3. Follow the setup wizard
4. Save your backup codes in a secure location

## Supported Authenticator Apps
- Google Authenticator
- Authy
- 1Password
- Microsoft Authenticator

## Lost Access to 2FA Device
1. Use one of your 10 backup codes (shown during setup)
2. If you have no backup codes, contact support with photo ID verification

## Disabling 2FA
Not recommended, but possible under **Settings > Security > 2FA > Disable**.
You must authenticate with 2FA to disable it.

## Organization-Enforced 2FA
Team owners can require 2FA for all members under **Settings > Team > Security > Require 2FA**.
Members without 2FA will be prompted to set it up on next login.

## Backup Codes
- 10 codes are generated at setup
- Each code can only be used once
- Regenerate new codes anytime (invalidates old ones)
