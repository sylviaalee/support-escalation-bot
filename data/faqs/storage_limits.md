# Storage and File Limits

## Storage Limits by Plan
- **Free**: 5 GB
- **Pro**: 100 GB
- **Business**: 1 TB
- **Enterprise**: Unlimited (fair use policy applies)

## File Size Limits
- Maximum single file upload: 2 GB
- Maximum files per project: 10,000

## Supported File Types
All file types are supported for storage. Processing (preview, indexing) is available for: PDF, DOCX, XLSX, images (PNG/JPG/GIF/WebP), video (MP4/MOV), audio (MP3/WAV).

## Checking Your Usage
**Settings > Storage** shows your current usage and breakdown by project.

## Exceeding Limits
If you exceed your storage limit:
- Uploads are blocked until you free up space or upgrade
- Existing files are not deleted
- You receive email warnings at 80%, 90%, and 100%

## Deleting Files
Deleted files go to **Trash** and are permanently removed after 30 days. Empty trash early under **Settings > Storage > Trash**.
